> Telegram-бот: персональный нумерологический разбор и прогноз по дате рождения с дифференциацией по тарифам и выдачей премиального PDF. Вся логика и тексты — перенос Excel-программы «МАТРИЦА» (50 листов, ~6 000 формул). **Источник истины — Excel: бот выдаёт ровно то, что считает книга** (нулевая толерантность к расхождениям на контрольных примерах).

## Проблема

Нумерологический разбор живёт в гигантском Excel-файле. Им нельзя пользоваться массово, продавать и автоматизировать: нет меню, оплаты, анкеты, дифференциации услуг, выдачи отчётов и истории клиентов.

## Решение

Продакшн-бот: пользователь выбирает тариф → оплачивает → заполняет короткую анкету → получает персональный отчёт (PDF/текст). Плюс интерактивный **прогноз на выбранный день**. Каждый параметр сопровождается **полным текстовым описанием**, извлечённым из листа «текст» Excel (не выдумано) и привязанным к рассчитанному значению (VLOOKUP-модель: меняется значение → меняется описание).

## Возможности

- **7 тарифов с дифференциацией по полям** — каждый тариф показывает ровно свой набор блоков (см. таблицу ниже). Реализовано выбором по атомам (`core/numerology/tariffs.py`), а не по грубым секциям.
- **Полные описания всех параметров** из Excel: ЧЖП (предназначение / недостатки / профессии / цвет удачи / талисман), число души (10 полей), персональное число года, кодировки, уровень сознания, код поведения.
- **Каждый код идёт с описанием** (в РАСЧЕТ ячейка-число ссылается на описание): «Код жизни» → **График кода жизни** активной цифры за текущий возраст (лист 19, `текст!B212`); «Жизненные силы» и «Доступ к деньгам» → порог-описание из формулы `AN15`/`AN14` (число + текст).
- **Кодировка «до/после 35 лет»** — два кода (`1-код` и `2-код`) с описаниями (методика сверена по формулам `calc!E18…E21`).
- **Прогноз по годам (1 / 5 лет)** — на каждый год своё описание: итог года (что ждёт / можно / нельзя), нулевой период (Солнце = 0), график кода жизни. Луна и Солнце по месяцам — на все годы прогноза (1 или 5 лет).
- **Прогноз на день** (меню «📅 Прогноз на день» / `/day`) на любую дату: ЧПД + трактовка, личный год/месяц, статус биоритма.
- **Визуальные графики** в PDF (как в книге): «График жизненной энергии» (6 цифр кода жизни), «График жизненных сил по годам» (энергокривая), «Энергетический график на текущий год» (помесячно) — серии сверены с книгой 1:1.
- **Полная психоматрица 17 качеств** (квадрат Пифагора + 8 производных: самооценка, целеустремлённость, таланты, духовно-материальный баланс и др.).
- **Премиальный PDF** (reportlab, PT Serif/Sans, квадрат Пифагора) + текстовая выдача с разбивкой по лимиту Telegram.
- **Платежи и история** — Telegram Stars (идемпотентность, возврат), «Мои расчёты», админка.

> Оплата через Telegram Stars сейчас **в тестовом режиме** (`PAYMENT_IMITATION=true`): бот имитирует успешный платёж для проверки функционала. Боевой режим включается `PAYMENT_IMITATION=false`. Прайс — в тенге (доллар показывается справочно).

## Архитектура

```
Telegram ──► bot/ (aiogram: меню, каталог, FSM-анкета, оплата, прогноз на день)
                   │
                   ▼
            core/numerology/  (расчёт = перенос листов Excel; matrix.py — golden)
                   │   tariffs.py — выбор блоков по полям (атомам)
                   │   daily.py   — прогноз на выбранную дату
                   ▼
            core/numerology/report.py (сборка + лукап трактовок по значениям)
                   │  ◄── core/content/ (тексты из листов «текст», 1–36)
                   ▼
            core/render.py / core/pdf.py ──► PostgreSQL (orders/payments/results)
```

Подробнее: [docs/architecture.md](docs/architecture.md). Карта извлечения текстов из Excel и сверка методик — [docs/content-extraction-map.md](docs/content-extraction-map.md).

## Quick Start

```bash
git clone https://github.com/m34959203/numerology-bot.git
cd numerology-bot
cp .env.example .env          # заполнить BOT_TOKEN и DATABASE_URL
python -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"
alembic upgrade head
python -m bot.main
```

Деплой в Docker (бот + Postgres) — [docs/deploy.md](docs/deploy.md). Инструкция и troubleshooting — [docs/setup.md](docs/setup.md).

## Стек

- **Бот:** Python 3.12+ · aiogram 3.x
- **БД:** PostgreSQL · SQLAlchemy 2 (asyncpg) · Alembic
- **Расчёты:** чистый Python + `python-dateutil` (без pandas/numpy)
- **PDF:** reportlab (бандл-шрифты PT Serif/Sans, fallback DejaVu)
- **Конфиг:** pydantic-settings (`.env`)
- **Тесты/линт:** pytest · ruff · black
- **Хостинг:** VPS, Docker Compose (polling)
