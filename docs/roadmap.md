# Roadmap (этапы разработки)

Последовательность обязательна. Каждый этап завершается прогоном тестов сверки с Excel.

## Этап 1 — Каркас + оплата (MVP-инфраструктура)
Бот, меню, каталог, FSM-анкета, оплата Telegram Stars, БД, заглушка выдачи.
**Готово, когда:** пользователь выбирает услугу → платит → заполняет анкету → получает заглушку.

## Этап 2 — Психоматрица + биоритмы
Перенос листов `calc` (квадрат Пифагора) и `Bio` (биоритмы) + тексты. Первая реальная выдача.
**Готово, когда:** `test_psychomatrix` и `test_biorhythm` зелёные на контрольных примерах.
- [x] Психоматрица (`calc`) — реализована, golden-тест 01.01.2000 зелёный.
- [x] Биоритмы (`Bio`) — реализованы, golden-тест зелёный (благоприятные/критические/травмоопасные дни 1:1 с книгой).
- [x] Тексты-трактовки (`текст`, 1–36) — выгружены, подключены в `report.py`, сверены.
- [x] **Полный реверс-инжиниринг всей книги** (Matr, Луна/Солнце, ЧПМ, сборка РАСЧЕТ) → `docs/excel-analysis.md`.

## Этап 3 — Коды и базовые вычисления
Полных лет, прожито дней, код жизни/человека/души, ЧЖП, финансовый код, счастливые числа (листы `РАСЧЕТ`, `Calculation`, `Lists`).
- [x] Коды от даты (`codes.py`): духовный уровень, код человека, счастливые числа,
      код жизни, фин-код, доступ к деньгам, жизненные силы, жизненная задача,
      полных лет, прожито дней — сверены на 3 эталонах (golden).
- [ ] Опасный возраст (нужна годовая таблица Солнца из Matr).
- [ ] Число имени / карма имени — после ФИО-эталонов (значения от имени).

## Этап 4 — Матрица (ядро `Matr`)
Полный реверс-инжиниринг листа `Matr` (4715 формул). Самый объёмный блок.
- [x] Прогноз на 5 лет (`matrix.py`): год, возраст, персональное число года + текст,
      Луна, Солнце, «год на +/−» + текст — сверены с эталоном (golden). В `report.py`.
- [x] Судьбоносные события (Z +/−) — цепочка aₙ₊₁=aₙ+Σцифр(aₙ); сверено на 2 примерах.
- [x] Энергетический потенциал (J) — энергокривая (опоры = первые 5 цифр кода жизни,
      линейная интерполяция /7); сверено.
- [x] Опасный возраст — первый возраст ≥ текущего с Солнцем=0; сверено (ex1=25, жен.=нет).
- [ ] 12-летний цикл перерождения (AF) — Matr BG/CH-ряды (единственный остаток прогноза).

## Этап 5 — Луна и Солнце + ЧПМ
Оставшиеся методики (`луна и солнце`, `ЧПМ`).

## Этап 6 — Полировка
PDF-выдача (если нужна), админка, нагрузочная проверка, деплой, приёмочное тестирование.

---

## Критерии приёмки

- [ ] Путь: меню → услуга → оплата → анкета → отчёт.
- [ ] Платёж идемпотентен; повторная выдача без новой оплаты невозможна.
- [ ] Возврат звёзд через админа.
- [ ] Каждая методика совпадает с Excel на всех контрольных примерах.
- [ ] Отчёт корректно разбивается по 4096 / формируется в PDF.
- [ ] «Мои расчёты» возвращает купленные отчёты.
- [ ] Секреты в `.env`; репозиторий чист.
- [ ] README с инструкцией запуска и деплоя.
