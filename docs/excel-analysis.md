# Реверс-инжиниринг Excel — рабочие заметки

Источник истины: `docs/ПРОГРАММА МАТРИЦА на русском.xlsx` (50 листов).
Здесь — восстановленные алгоритмы по листам. Пополняется по этапам.

> **Ключевой приём.** В книге сохранены кэшированные значения для примера-входа
> **C12 = 01.01.2000** (ФИО и даты родителей пусты). Эти значения = готовые
> golden-эталоны для методик, зависящих только от даты (психоматрица, биоритмы,
> часть кодов). Читаются через `openpyxl(data_only=True)`.

## Лист РАСЧЕТ — вход и сборка отчёта

**Вход** (ячейки): `N9` — дата рождения; `N16 = TODAY()`. Метки ФИО/дат — в колонке `C`
(C6 Фамилия, C7 Имя, C8 Отчество, C9 дата рожд., C11 мать, C12 отец, **C14 девичья
фамилия — подпись по-казахски** «Тұрмысқа шыққанға дейінгі тегі» → интерфейс двуязычный).

**Выход** агрегируется ссылками на другие листы:
- блок «Вычисления» (AC/AN) ← `Matr` (код человека/жизни, фин-код, опасный возраст…);
- блок «Психоматрица» (K/O/S/W) ← `calc`;
- блок «Дни» (BD) ← `Bio` (благоприятные/критические/травмоопасные);
- блок «Прогноз на 5 лет» (B52+) ← `Matr` (персональное число года, циклы, Луна/Солнце).

## Лист calc — психоматрица ✅ реализовано (`core/numerology/psychomatrix.py`)

`C12 = РАСЧЕТ!N9`. Алгоритм квадрата Пифагора:

1. **Цифры даты** (D16:K16): день(2 цифры), месяц(2, добивка 0 при однозначном), год(4).
2. **Рабочие числа**: `E18` = сумма 8 цифр; `E19` = свод E18; `E20 = E18 − 2×(первая
   ненулевая цифра дня)` (спец-ветка при <0); `E21` = свод E20.
   Строка `U16 = concat(E18,E19,E20,E21)`, её цифры идут в пул.
3. **Пул** = цифры даты + цифры рабочих чисел. `COUNTIFS` вхождений 1..9 (нули не в счёт)
   = клетки квадрата (U19:W21).
4. **Качества** (РАСЧЕТ): повтор-цифры клеток (U24=count×digit) и суммы рядов/столбцов
   (целеустремлённость=c1+c4+c7, самооценка=c1+c2+c3, и т.д.). 0 → «нет».
5. **Скаляры**: число души `F1=DAY`; уровень сознания `C1=день+месяц+Σцифр года`;
   ЧЖП `J19=VLOOKUP(E19)` = свод `((n-1)%9)+1` (мастер-числа НЕ сохраняются!);
   код жизни `C14 = день×месяц×год`, добивка нулями до 6 знаков;
   код поведения `D13 = VLOOKUP(дата)` — таблица 13 «кодов» по диапазонам дат
   → выгружена в `core/content/data/behavior_codes.json` (ключ `MM-DD`).

Golden-эталон 01.01.2000 → `tests/fixtures/psychomatrix_2000_01_01.json` (тест зелёный).

## Лист Bio — биоритмы ⏳ следующий (Этап 2)

37677 строк = по дню от даты рождения (`B2=РАСЧЕТ!N9`, `B3=B2+1`, …).
- `I/J/K` — фазовые аккумуляторы циклов **физ=23 / эмо=28 / интел=33** дней
  (инкременты `AF$3`, `AC$15`, `AC$20`; перенос через длину цикла). `F/G/H = ROUND(I/J/K)`.
- `L/M/N` — флаги «высокой» зоны (пик физ при F=12 и т.п.); `O = L+M+N`.
- `P` — критические дни (совпадение пиков 11/14/16); `R` — классификация зоны.
- `C/D/E` — текстовые трактовки (IF-цепочки по F/G/H в колонку Y).
- РАСЧЕТ тянет готовые списки дат из колонок `AE/AF` (благоприятные/критические) и
  `AZ/BA` (травмоопасные) — это агрегаты выбранных дат в окне прогноза.
